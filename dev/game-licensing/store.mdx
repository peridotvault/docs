## Overview

Game Store handles purchase flow.

Responsibilities:

- Validate game
- Process payment
- Mint license (via PGC-1)

Game Store does NOT store metadata or governance.

---

## Dependencies

- Registry
- PGC-1
- Factory (optional)

## Flow

1. Fetch game from Registry

const game = Registry.getGame(gameId)

2. Validate

- game must be `approved`

3. Process payment

4. Mint license

PGC1.mintLicense(msg.sender, gameId, 0)
